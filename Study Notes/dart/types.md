# Dart 타입 (Types)

Dart는 **정적 타입** 언어이다. 변수 선언 시 타입을 명시하거나, `var`로 타입 추론을 사용할 수 있다.

## 숫자 (Number)

```dart
int a = 10;       // 정수
double b = 3.14;  // 실수
num c = 42;       // int와 double 모두 가능
```

| 타입 | 설명 | 예시 |
|------|------|------|
| `int` | 정수 | `1`, `-5`, `100` |
| `double` | 실수 (부동소수점) | `3.14`, `-0.5` |
| `num` | int와 double을 모두 포함하는 상위 타입 | `42`, `3.14` |

---

## 문자열 (String)

```dart
String s1 = '작은따옴표';
String s2 = "큰따옴표";
String s3 = '이름: ${s1}, 나이: ${a}';  // 문자열 보간법
```

- 작은따옴표(`'`)와 큰따옴표(`"`) 모두 사용 가능
- **문자열 보간법**: `${변수}` 또는 `$변수` 형태로 문자열 안에 변수를 삽입할 수 있다

---

## 불리언 (Boolean)

```dart
bool flag = true;
```

- `true` 또는 `false` 두 가지 값만 가진다
- 조건문(`if`, `while` 등)에서 주로 사용

---

## 리스트 (List)

```dart
List<int> numbers = [1, 2, 3, 4, 5];
var fruits = ['사과', '바나나', '포도'];
```

- 순서가 있는 데이터 집합 (배열)
- `List<타입>`으로 제네릭 타입 지정 가능
- `var`를 사용하면 타입 추론으로 자동 결정

---

## 맵 (Map)

```dart
Map<String, int> scores = {
  '수학': 90,
  '영어': 85,
  '국어': 95,
};
```

- **키-값 쌍**으로 이루어진 데이터 구조
- `Map<키타입, 값타입>` 형태로 선언
- 키는 유일해야 하며, 값은 중복 가능

---

## 셋 (Set)

```dart
Set<String> tags = {'dart', 'flutter', 'dart'};  // 'dart' 하나만 저장됨
```

- **중복을 허용하지 않는** 데이터 집합
- 같은 값을 여러 번 추가해도 하나만 저장됨
- 순서를 보장하지 않음
